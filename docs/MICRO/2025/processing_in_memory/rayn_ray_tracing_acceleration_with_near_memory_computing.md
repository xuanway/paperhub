## 一句话总结

> 该工作面向 Processing-In-Memory 场景，围绕 Ray Tracing Acceleration with Near-memory Computing 提出系统化优化方案，重点涉及 RayN、Ray、Tracing 等关键技术点。

## 方法简述

- 以 RayN 为核心切入点，构建针对 Processing-In-Memory 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 优化内存访问、分配或一致性路径。

## 主要结果

- 降低内存瓶颈并提升系统效率。
- 目标是降低时延并提升吞吐/可扩展性。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
