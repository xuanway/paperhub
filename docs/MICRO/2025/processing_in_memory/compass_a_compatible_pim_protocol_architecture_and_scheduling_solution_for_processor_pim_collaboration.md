## 一句话总结

> 该工作面向 Processing-In-Memory 场景，围绕 ComPASS、Compatible 相关关键问题 提出系统化优化方案，重点涉及 ComPASS、Compatible、PIM 等关键技术点。

## 方法简述

- 以 ComPASS 为核心切入点，构建针对 Processing-In-Memory 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 利用近数据计算减少数据搬运。

## 主要结果

- 在能效与吞吐之间取得更优平衡。
- 在真实系统落地时兼顾实现复杂度与工程可维护性。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
