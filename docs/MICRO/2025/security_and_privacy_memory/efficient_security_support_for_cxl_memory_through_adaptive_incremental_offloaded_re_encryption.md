> 该工作面向 Security and Privacy - Memory 场景，围绕 Security、Support 相关关键问题 提出系统化优化方案，重点涉及 Security、Support、CXL 等关键技术点。

## 方法简述

- 以 Security 为核心切入点，构建针对 Security and Privacy - Memory 工作负载的优化路径。

- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
