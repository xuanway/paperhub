## 一句话总结

> 该工作面向 GPUs 场景，围绕 Security and Performance Implications of GPU Cache Eviction Priority Hints 提出系统化优化方案，重点涉及 Security、Performance、Implications 等关键技术点。

## 方法简述

- 以 Security 为核心切入点，构建针对 GPUs 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 在 GPU 执行与调度路径上引入针对性优化。

## 主要结果

- 提升并行执行效率与资源利用率。
- 降低访存开销并改善性能稳定性。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
