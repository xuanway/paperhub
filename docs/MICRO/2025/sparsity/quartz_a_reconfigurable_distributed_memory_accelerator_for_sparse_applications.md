## 一句话总结

> 该工作面向 Sparsity 场景，围绕 A Reconfigurable, Distributed Memory Accelerator for Sparse Applications 提出系统化优化方案，重点涉及 Quartz、Reconfigurable、Distributed 等关键技术点。

## 方法简述

- 以 Quartz 为核心切入点，构建针对 Sparsity 工作负载的优化路径。

- 降低内存瓶颈并提升系统效率。

- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
