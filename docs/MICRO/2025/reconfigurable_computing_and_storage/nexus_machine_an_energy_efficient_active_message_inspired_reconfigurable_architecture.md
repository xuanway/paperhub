## 一句话总结

> 该工作面向 Reconfigurable Computing and Storage 场景，围绕 An Energy-Efficient Active Message Inspired Reconfigurable Architecture 提出系统化优化方案，重点涉及 Nexus、Machine、Energy-Efficient 等关键技术点。

## 方法简述

- 以 Nexus Machine 为核心切入点，构建针对 Reconfigurable Computing and Storage 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 将能耗建模纳入优化目标。

## 主要结果

- 降低单位任务能耗。
- 目标是降低时延并提升吞吐/可扩展性。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
