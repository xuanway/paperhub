## 一句话总结

> 该工作面向 CXL Storage 场景，围绕 Formalising CXL Cache Coherence 提出系统化优化方案，重点涉及 Formalising、CXL、Cache 等关键技术点。

## 方法简述

- 以 Formalising CXL Cache Coherence 为核心切入点，构建针对 CXL Storage 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 针对缓存层次与数据复用策略进行改进。

## 主要结果

- 降低访存开销并改善性能稳定性。
- 在真实系统落地时兼顾实现复杂度与工程可维护性。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
