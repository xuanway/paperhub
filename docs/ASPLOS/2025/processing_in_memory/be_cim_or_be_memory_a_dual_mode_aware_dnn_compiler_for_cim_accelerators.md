> 该工作面向 Processing in Memory 场景，围绕 A Dual-mode-aware DNN Compiler for CIM Accelerators 提出系统化优化方案，重点涉及 CIM、Memory、Dual-mode-aware 等关键技术点。

## 方法简述

- 以 Be CIM or Be Memory 为核心切入点，构建针对 Processing in Memory 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。

## 主要结果

- 降低内存瓶颈并提升系统效率。

- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
