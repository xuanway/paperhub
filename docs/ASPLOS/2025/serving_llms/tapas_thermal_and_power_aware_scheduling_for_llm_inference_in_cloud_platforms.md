## 一句话总结

> 该工作面向 Serving LLMs 场景，围绕 Thermal- and Power-Aware Scheduling for LLM Inference in Cloud Platforms 提出系统化优化方案，重点涉及 TAPAS、Thermal-、Power-Aware 等关键技术点。

## 方法简述

- 以 TAPAS 为核心切入点，构建针对 Serving LLMs 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 结合大模型推理/训练链路进行系统级优化。

## 主要结果

- 突出吞吐、时延与成本的综合改进。
- 改善性能功耗比。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
