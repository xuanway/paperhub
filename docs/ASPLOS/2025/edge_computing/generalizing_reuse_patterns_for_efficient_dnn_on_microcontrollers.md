## 一句话总结

> 该工作面向 Edge Computing 场景，围绕 Generalizing Reuse Patterns for Efficient DNN on Microcontrollers 提出系统化优化方案，重点涉及 Generalizing、Reuse、Patterns 等关键技术点。

## 方法简述

- 以 Generalizing 为核心切入点，构建针对 Edge Computing 工作负载的优化路径。
- 从算法、编译或体系结构层面进行联合设计，减少关键瓶颈。
- 通过模块化设计保持方案可迁移性与工程可落地性。

## 主要结果

- 目标是降低时延并提升吞吐/可扩展性。
- 在真实系统落地时兼顾实现复杂度与工程可维护性。
- 方法具备与现有软硬件栈集成的潜力，适用于后续扩展验证。
